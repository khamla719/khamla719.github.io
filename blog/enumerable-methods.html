Enumerable Methods: Map

December 14, 2015

The enumerable method "map" is helpful little tool that can make finding things much easier. The "map" enumerator allows you to perform an action for each element in a given array. The results are returned in a new array so you can see exactly what happened to each element.

This can come in handy for countless situations when coding. In the example above, let's say you have an array of numbers 1 -10. Let's say you wanted to increment that array by 5 for each number. You could use "array.map {|i| i + 5}". Or, if you wanted to know which numbers in the array are even, you could use "array.map {|i| i.even?}". This will show True if the number is even and False if not. You can probably think of many more things to do with "map" and I'm sure I will use it plenty as my skills progress in Ruby
